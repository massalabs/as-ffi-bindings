fn read(&self, memory: &Memory, store: &impl AsStoreRef) -> Result<T>

Read the value contained in the given memory at the current pointer offset.

§Return

A result with an Error if for some reason the binding failed to read the offset returned. It can be a cast error for example.

Otherwise, a success with the value red at the point.

§Example
ⓘ
let get_string = instance
    .exports
    .get_native_function::<(), StringPtr>("getString")?;
let str_ptr = get_string.call()?;
let string = str_ptr.read(memory)?;

fn size(&self, memory: &Memory, store: &impl AsStoreRef) -> Result<u32>

Read the size as indicated in the AssemblyScript object header

§Return

A result with an Error if for some reason the binding failed to read the offset returned. It can be a cast error for example.

Otherwise, a success with the size.

§Example
ⓘ
let get_string = instance
    .exports
    .get_native_function::<(), StringPtr>("getString")?;
let str_ptr = get_string.call()?;
let size: u32 = str_ptr.size(memory)?;

Dyn Compatibility§

This trait is not dyn compatible.

In older versions of Rust, dyn compatibility was called "object safety", so this trait is not object safe.
